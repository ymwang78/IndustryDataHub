
Version 1.06

支持4种文件格式
 
    1. TaiJiMPC.exe 创建的工程文件 *.ojp

    2. Matlab文件格式定义的状态文件
       Afr        # 状态（系数）矩阵
       Bfr        # 控制（输入）矩阵
       Cfr        # 观测（输出）矩阵
       Dfr        # 前馈（输入输出）矩阵
       Uname      # MV和DV名称
       MeanU      # MV和DV初始值(平均值)
       STDu       # MV和DV的standard deviation
       MV_DV      # MV和DV指示标志, 0值为MV, 1为DV
       Yname      # CV名称
       MeanY      # CV初始值(平均值)
       STDy       # CV的standard deviation
       DelayVec   # CV的时延
       CV_INT     # CV的积分指示标志，0非积分，1为积分

       例子文件：TaiJiOPCSim\model\LargeModel.mat

    3. Text文件格式定义的状态文件
       例子文件：TaiJiOPCSim\model\ShellModel_sp.ini

    4. Text文件格式定义的传递函数文件
       例子文件：TaiJiOPCSim\model\ShellModel_tf.ini
